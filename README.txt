# E-Commerce System with AI Phone Finder

This project is a complete e-commerce web application designed to provide users with a smooth online shopping experience and give administrators full control over products, orders, users, and website content.

The system includes a modern storefront, product catalog, user authentication, wishlist, checkout flow, support pages, and an admin dashboard. One of the main features of the project is the **AI Phone Finder**, which helps users choose the most suitable smartphone based on their needs, budget, and preferences.

---

## Project Overview

The goal of this project is to build a practical e-commerce platform that is not only focused on selling products, but also helps users make smarter purchasing decisions.

Many users struggle when choosing a phone because there are many models, technical specifications, and price ranges. To solve this problem, the project includes an AI-based recommendation feature that guides users and suggests phones in a simple and personalized way.

## AI Phone Finder Feature

The **AI Phone Finder** is the most important smart feature in this project.

Instead of asking the user to manually compare many phones, the system asks simple questions about the user's needs, such as:

- Budget range
- Preferred brand
- Camera importance
- Gaming performance
- Battery life
- Storage needs
- General usage style

Based on these answers, the system analyzes the user's preferences and recommends suitable phones.

The AI feature is designed to make the shopping experience more helpful and personalized. It works as a smart assistant that explains why a specific phone is suitable for the user, instead of only showing random products.

---

## How the AI Recommendation Works

The AI Phone Finder uses a recommendation logic that compares the user's answers with available phone specifications.

The system considers important factors such as:

- Price compatibility
- Performance level
- Camera quality
- Battery capacity
- Storage and memory
- User priorities

After analyzing these factors, the system generates a recommendation with a clear explanation. This helps the user understand the reason behind the suggested phone.

The AI module improves the decision-making process and makes the website more interactive compared to a normal e-commerce store.
